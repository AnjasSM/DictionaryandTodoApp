# TodoApp
todo app

### Requirment

- NodeJS v8+
- Android Studio
- Java SE Development Kit (JDK)
- React Native CLI

### How To Run

How to Install :

```sh
$ git clone https://github.com/AnjasSM/DictionaryandTodoApp.git`` 
$ cd DictionaryandTodoApp 
$ npm install
$ react-native run-android
```
